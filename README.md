# 🏠 Smart Home Automation Dashboard

A Smart Home Automation and Security Monitoring Dashboard developed using Streamlit and Python.

## Features

- Appliance Control (Light & Fan)
- Security System Monitoring
- Motion Detection Simulation
- Buzzer Alert Simulation
- LCD Display Simulation
- LDR Sensor Monitoring
- Real-Time Dashboard Interface

## Technologies Used

- Python
- Streamlit
- Arduino Project Concept
- IoT Simulation

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
